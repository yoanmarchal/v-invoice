## Introduction

Install *v-invoice*:

```bash
# Npm
npm i --save yoanmarchal/v-invoice
npm i --save vue2-filters
# Yarn
yarn add yoanmarchal/v-invoice vue2-filters
```

Then use the component in an Vuejs project:
